테토에겐(TETO-EGEN) 개인정보처리방침

시행일: 2025년 1월 24일

1. 개인정보의 수집 및 이용 목적

테토에겐 앱은 사용자의 개인정보를 수집하지 않습니다. 모든 테스트 결과와 응답은 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 수집하는 개인정보 항목

테토에겐은 다음과 같은 개인정보를 수집하지 않습니다:

  • 이름, 이메일, 전화번호 등의 개인 식별 정보
  • 위치 정보
  • 결제 정보
  • 기기 고유 식별자

3. 테스트 데이터 처리

  • 모든 테스트 응답과 결과는 사용자의 기기에만 로컬로 저장됩니다
  • 앱을 삭제하면 모든 데이터가 완전히 제거됩니다
  • 테스트 결과 공유 시, 사용자가 직접 선택한 방법으로만 공유됩니다

4. 광고 및 분석

테토에겐은 Google AdMob을 통해 광고를 표시합니다. Google AdMob은 광고 표시를 위해 다음 정보를 수집할 수 있습니다:

  • 광고 ID (개인 식별 불가)
  • 광고 상호작용 데이터
  • 기기 정보 (OS 버전, 앱 버전)

Google의 개인정보처리방침: https://policies.google.com/privacy

5. 데이터 보안

테토에겐은 사용자 데이터를 수집하거나 저장하지 않으므로, 데이터 유출의 위험이 없습니다. 모든 테스트 데이터는 사용자의 기기에서만 처리됩니다.

6. 아동의 개인정보 보호

테토에겐은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만의 아동은 부모님의 동의 하에 앱을 사용해야 합니다.

7. 개인정보처리방침의 변경

본 개인정보처리방침이 변경될 경우, 앱 내 공지사항을 통해 사전에 알려드립니다.

8. 연락처

개인정보처리방침에 대한 문의사항이 있으시면 아래로 연락주시기 바랍니다:

  • 이메일: support@jobby-time.com
  • 앱 이름: 테토에겐 (TET-EGEN)

9. 권한 사용

테토에겐은 다음과 같은 권한을 요청합니다:

  • 인터넷 접근: 광고 표시를 위해 필요
  • 네트워크 상태 확인: 광고 로드 최적화를 위해 필요

10. 사용자의 권리

테토에겐은 개인정보를 수집하지 않으므로, 개인정보 열람, 정정, 삭제 요청이 해당되지 않습니다. 앱 사용에 대한 모든 데이터는 사용자가 직접 관리할 수 있습니다.
